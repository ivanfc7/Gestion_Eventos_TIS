# Gestion_Eventos
Aplicación web desarrollada como proyecto académico. Permite la publicación y gestión de eventos competitivos, así como la inscripción de participantes. Incluye panel administrativo para la creación de eventos y editor de afiches.

## Tecnologías Utilizadas
- Backend: Laravel
- Frontend: Blade + JavaScript
- Estilos: Bootstrap
- Base de datos: MySQL
- Gestión de dependencias: Composer
- Control de versiones: Git
